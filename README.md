## Description

A repository documenting my progress, findings, and lessons learned while completing TryHackMe challenges, along with my efforts to improve my write-up skills.


## Config

Some tryhackme labs will require that you connect to their lab machine via your own machine. tryhackme offers two ways to connect to their labs.
1. Using their own "attackbox" web-based machines that is easy to access however you only get access to their machines 60 min/day.
2. Using your own operating system and connecting into tryhackme's network through openvpn.
